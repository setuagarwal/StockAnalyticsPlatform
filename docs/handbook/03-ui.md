# Stock Analytics Platform - UI Specification

## Purpose

Define the Version 1 user interface for the Stock Analytics Platform.

The application uses a single, workspace-centric desktop interface focused on stock analysis.

---

# Main Layout

```text
+--------------------------------------------------------------------------------+
| Search | Exchange | Timeframe | Indicators | Workspace | Alerts | Settings    |
+--------------------------------------------------------------------------------+
| Left Panel        | Main Chart Area                              | Right Panel |
| Watchlist         | Candlestick Chart                            | AI Insights |
| Search Results    | Indicators                                   | News        |
| Workspaces        | Drawings                                     | Patterns    |
+-------------------+----------------------------------------------+-------------+
| Bottom Panel: Notes | Alerts | Logs | Status                                   |
+--------------------------------------------------------------------------------+
```

# Position & Target Tracking

Every analysed stock can be in one of three states.

## 1. Untracked

Default state.

No tracking information is stored.

---

## 2. Owned Position

Fields:

- Average Buy Price
- Quantity
- Target Sell Price (optional)
- Stop-loss (optional)
- Notes (optional)

The UI should always display this information whenever the stock is opened.

---

## 3. Watch Position

Fields:

- Target Buy Price
- Target Buy Price Range (optional)
- Optional Stop-loss
- Notes

The application monitors these targets and generates alerts when conditions are met.
